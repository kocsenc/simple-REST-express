# simple-REST-express
Very simple REST example using node.js and express.
Meant to be an educational introductory tool to REST Interfaces.

# Get started
1. Clone this repo
1. `npm install`
1. `node server.js`

## Architecture
**Data Store**: File

**body-parser (middleware)**: URL encoded

## Endpoints
You can `GET`, `GET/:id` and `POST` on `/users`
